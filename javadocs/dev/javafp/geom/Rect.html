Class Rect

java.lang.Object
dev.javafp.val.ImValuesImpl
dev.javafp.geom.Rect
All Implemented Interfaces:
HasTextBox, Values, Serializable
public class Rect extends ImValuesImpl

A Rectangle.

A rectangle with its north-west point origin and its south-east point corner so its size is corner - origin

This is used to layout widgets (and other things).

 [this, this.moveBy(offset), this.moveBy(offset*2).., this.moveBy(offset*count) ]
