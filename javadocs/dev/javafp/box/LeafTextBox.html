public class LeafTextBox extends AbstractTextBox

A text box that does not contain any other text boxes.

It consists of 1 or more lines of text - ie Strings.

Any tab characters will be expanded to spaces assuming a tab width of 4.

There will be no newline characters in any of the lines.

If any of the characters used to create the box are ISO control characters, they will be converted using transformISOControlChars(String)

A LeafTextBox on the empty string has width 0 and height 1

    • centred

      public static LeafTextBox centred(String text, int width)

      A LeafTextBox containing text centred in width .

      The centering is done using space characters.

      Any tabs in text are expanded assuming a tab width of 4 using TextUtils.detab(int, String)

      text is not trimmed.

      If any line of the text, l , is such that width <= l.size() then this line is right trimmed to length width

      Otherwise, let d = width - text.size()

      if d is not even then there will be (d - 1)/2 spaces on the left and (d - 1)/2 + 1 spaces on the right

      Examples: 

      
 centred("abc", 7) => "  abc   "
 centred("abc", 2) => "ab"
 centred("abc", 4) => "abc "
 centred("abc\nde", 14) =>
 "     abc      "
 "      de      "
