    • equals

      public boolean equals(Object other)

      Compares other with this for equality. Returns true if and only if the specified object is also an ImSortedSet , both SortedSets have the same size, and: 

      
 this.containsAll(other)

      In other words, two SortedSets are defined to be equal if they contain equal elements.

      In other other words - this is very similar to the standard Java SortedSet equals .
