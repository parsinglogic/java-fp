    • getBoxFrom

      public static AbstractTextBox getBoxFrom(Object thing)

      Get a text box from thing

      It uses various tricks to do this.

      If thing is a AbstractTextBox then it just returns thing

      If thing is a HasTextBox then it just returns thing.getTextBox()

      If thing is an array then it converts the array to an ImList and calls getBoxFromList on the list.

      To convert the array to an ImList , if the array contains primitives it uses ImList.onPrimitiveArray(Object) otherwise it uses ImList.on(Object[])

      Otherwise it creates a LeafTextBox on the result of thing.toString()
