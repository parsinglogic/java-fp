Serializable
public class SvgTransform extends Object implements Serializable

transform="matrix(a,b,c,d,e,f)"

The matrix is this - using homogeneous coordinates: 


 a c e
 b d f
 0 0 1

Applying a matrix to a point 


            x
            y
            1

 a c e      ax + cy + e
 b d f      bx + dy + f
 0 0 1      1

translation by (e,f) 


 1 0 e      x + e
 0 1 f      y + f
 0 0 1      1

scaling by a in x and d in y 


 a 0 0       ax
 0 d 0       by
 0 0 1       1

Rotation about the origin by alpha anticlockwise: 


              x
              y
              1

 cos -sin 0   x*cos - x*sin
 sin  cos 0   x*sin + x*cos
 0    0   1   1

where cos = cos(alpha), sin = sin(alpha)
