Package dev.javafp.tree

Class ImTreeShapes

java.lang.Object
dev.javafp.tree.ImTreeShapes
public class ImTreeShapes extends Object

Consider the possible tree shapes of size 2: 


    *   *
   *      *

Consider the possible tree shapes of size 3: 


    *      *     *     *      *
   *      *     * *      *      *
  *        *            *        *

Consider the sizes of the root tree and its immediate child nodes: 


    3       3     3      3      3
   2 0     2 0   1 1    0 2    0 2

So, to generate shapes of size 3 we start with a root node r

and then we generate all trees with r as the root and

  1. a left child of size 2 and a right child of size 0 (nil)

  2. a left child of size 1 and a right child of size 1

  3. a left child of size 0 and a right child of size 2

We can generalise this and observe that,
to generate all possible tree shapes of size n, we generate tree shapes with sizes like this: 


      n        n                  n
     / \      / \     ...        / \
   n-1  0   n-2  1              0  n-1

We need to generate all possible tree shapes for the children as well. This can be done recursively of course
